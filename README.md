# Mathematical-Number-Concepts
All possible well-structured list of important mathematical/number concepts



1пёҸвғЈ Prime Number
Definition

A number greater than 1 that has exactly two divisors:
1
Itself

Logic
If no number between 2 and вҲҡn divides it вҶ’ it is prime.

Example:
2, 3, 5, 7, 11, 13вҖҰ



2пёҸвғЈ Composite Number
Definition

A number greater than 1 that has more than two divisors.

Example:
4 (1,2,4), 6 (1,2,3,6)



3пёҸвғЈ Even Number
Definition

Divisible by 2.

Logic
n % 2 == 0
n & 1 == 0

Binary Logic:
Last bit = 0



4пёҸвғЈ Odd Number
Definition

Not divisible by 2.

Logic
n % 2 != 0
n & 1 != 0
Binary: Last bit = 1



5пёҸвғЈ Armstrong Number (Narcissistic Number)
Definition

Sum of digits raised to power of number of digits equals the number itself.

Logic (3-digit example)

For 153:
1Ві + 5Ві + 3Ві = 153

Works for any number of digits.



6пёҸвғЈ Palindrome Number
Definition

Number that reads same forward and backward.

Example:
121, 1331

Logic
Reverse the number and compare with original.



7пёҸвғЈ Fibonacci Series
Definition

Each number is sum of previous two.

0, 1, 1, 2, 3, 5, 8, 13...

Formula
F(n) = F(nвҲ’1) + F(nвҲ’2)



8пёҸвғЈ Perfect Number
Definition

Sum of proper divisors equals the number.

Example:
6

Divisors:
1, 2, 3
Sum = 6



9пёҸвғЈ Strong Number
Definition

Sum of factorial of digits equals number.

Example:
145

1! + 4! + 5! = 145



рҹ”ҹ Automorphic Number
Definition

Square of number ends with the number itself.

Example:
5ВІ = 25
6ВІ = 36
25ВІ = 625



1пёҸвғЈ1пёҸвғЈ Harshad (Niven) Number
Definition

Divisible by sum of digits.

Example:
18
1+8 = 9
18 % 9 == 0



1пёҸвғЈ2пёҸвғЈ Neon Number
Definition

Sum of digits of square equals number.

Example:
9ВІ = 81
8+1 = 9



1пёҸвғЈ3пёҸвғЈ Spy Number
Definition

Sum of digits = Product of digits.

Example:
1124
Sum = 8
Product = 8



1пёҸвғЈ4пёҸвғЈ Duck Number
Definition

Contains zero but does not start with zero.

Example:
1023



1пёҸвғЈ5пёҸвғЈ Kaprekar Number
Definition

Square split into two parts whose sum equals original number.

Example:
45
45ВІ = 2025
20 + 25 = 45



1пёҸвғЈ6пёҸвғЈ Disarium Number
Definition

Sum of digits powered with their positions equals number.

Example:
135
1В№ + 3ВІ + 5Ві = 135



1пёҸвғЈ7пёҸвғЈ Happy Number
Definition

Repeatedly replace number by sum of square of digits.
If it ends in 1 вҶ’ Happy.

Example:
19 вҶ’ 82 вҶ’ 68 вҶ’ 100 вҶ’ 1



1пёҸвғЈ8пёҸвғЈ Abundant Number
Definition

Sum of proper divisors > number.

Example:
12
1+2+3+4+6 = 16 > 12



1пёҸвғЈ9пёҸвғЈ Deficient Number
Definition

Sum of proper divisors < number.



2пёҸвғЈ0пёҸвғЈ Circular Prime
Definition

All rotations of number are prime.

Example:
197
971
719
All prime.



2пёҸвғЈ1пёҸвғЈ Twin Prime
Definition

Prime numbers with difference of 2.

Example:
(11, 13), (17, 19)



2пёҸвғЈ2пёҸвғЈ Emirp Number
Definition

Prime number whose reverse is also prime and different.

Example:

2пёҸвғЈ3пёҸвғЈ Factorial
Definition

n! = n Г— (n-1) Г— (n-2) вҖҰ

Example:

рҹ”ў 2пёҸвғЈ4пёҸвғЈ GCD (Greatest Common Divisor)

Largest number dividing both numbers.

Logic:
Euclidean Algorithm.

рҹ”ў 2пёҸвғЈ5пёҸвғЈ LCM (Least Common Multiple)

рҹ”ў 2пёҸвғЈ6пёҸвғЈ Perfect Square

Example:

рҹ”ў 2пёҸвғЈ7пёҸвғЈ Perfect Cube
